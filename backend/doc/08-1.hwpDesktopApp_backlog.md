## 진행 현황 메모
- DesktopApp 메인 창에 Markdown 아티팩트 다중 선택 다운로드/저장 기능을 구현하고, 출력 경로 검증·파일명 정제·중복 방지 로직을 추가했습니다. (2025-01-17)
- API 클라이언트에 `/api/artifacts/:id/content` 호출 전용 메서드를 도입해 Markdown 콘텐츠를 안전하게 수신합니다.
- DesktopApp API 클라이언트에 Topic/Artifact DTO 기반 파서를 도입하고 UI 연동을 정비해 Phase 2의 DTO 정리 작업을 마무리했습니다. (2025-01-20)
- API 연결이 어려운 환경에서도 메인 UI를 체험할 수 있도록 데모용 Topic/Artifact 샘플 데이터를 추가하고 자동 대체 로직을 구성했습니다. (2025-01-20)
